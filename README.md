# Pinyin Sounds Training

Lightweight tool for training Chinese Pinyin listening and pronunciation.

## How it works
- Listen to the sound
- Type what you hear
- When correct, focus moves automatically
- Optional random order

## Features
- Keyboard-focused learning
- Instant repetition
- Minimal interface
- No installation required
- Works in browser
